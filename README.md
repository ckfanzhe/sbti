# SBTI Personality Test

> *Why take a personality test when you could take THE personality test?*

SBTI (**Silly Big Personality Test**) is the anti-MBTI. No four-letter codes, no Jungian archetypes, no career counseling brochures. Just 31 brutally honest questions that reveal which pile of human chaos you really are.

**Are you a CTRL** (The Controller — perfectionists who rest Ctrl+S over their entire existence)?

**Are you a SHIT** (The Cynic — gripes about everything but still opens Excel to get shit done)?

**Or worse... are you a DIO-S** (The Loser — congratulations, you've achieved Diogenes' Original Realism)?

---

### How It Works

1. Answer 31 questions (plus 2 bonus if you dare)
2. Get matched against 26 unique personality types across 15 dimensions
3. Receive a brutally accurate portrait of your magnificent disaster of a self

---

### Features

- **26 Distinct Types** — From "MUM" to "FUCK", each with its own lore
- **15 Dimensions** — S1 to So3, measuring everything from self-esteem to social awkwardness
- **Hidden Types** — Answer certain questions wrong and unlock the legendary DRUNK type
- **Shitpost-Grade Accuracy** — May or may not reflect actual personality
- **Multi-Language Support** — 中文 / English fully supported, with 日本語 and 한국어 planned

### Host It Yourself

This is a static site — host it anywhere. To deploy on **GitHub Pages**:

1. Fork or clone this repo
2. Go to Settings → Pages → Source → Select `main` branch and `/ (root)`
3. Enable GitHub Actions if prompted
4. Push to `main` — deployment is automatic
